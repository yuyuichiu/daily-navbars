Big O notation

To calculate the rate of how input size affects code execution time

Program run time factor is mainly noted with big-O notation to identify the root of the runtime problem

  • O(1) = Constant
  • O(log n) = Logarithm
  • O(n) = Linear
  • O(n log n) = Linearithmic (n in front of log)
  • O(n^2) = Polynomial
  • O(2^n) = Exponential
