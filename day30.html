Class in javascript

Components : Constructor, Get Method and Method

Constructor - run on initialization of Object, usually used for declaring class variables

    constructor(var1){ this.var1 = var1 }

Method: getter method & class method

get func(){ ... } => called by "class.func"

func(){ ... } => called by "class.func()"

Adding HTML object through javascript


Create the object (element and textnote)

var newTag = document.createElement("");

var newContent = document.createTextNote("Content to be inside");


Glue them together

newTag.appendChild(newContent);


Locate the parent & Insert

parent = getElementById("parentdiv")

parent.insertBefore(newTag, position)

Note: "position.nextSibling" if need to insert After (there is no insertAfter() method.)
