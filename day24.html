Pig Latin!

For each word, the first letter is put on the back with "ay", alone Punctuation marks are ignored

E.g. "Hello World !" => "elloHay orldWay !"

  • /regexp/.test(str) ==> Return true/false based on if regex found any match in string
  • str .trim() ==> Remove Unnecessary Whitespace from string. (e.g. " Hello " ==> "Hello")
  • Method: use split to separate words, then loop through each to add the correct letters to the result with modification at the end.

Dice Game

Roll five dice and see how much score you get!

Rules are as follow:

  • Three 1: 1000pt
  • Three 2: 200pt
  • Three 3: 300pt
  • Three 4: 400pt
  • Three 5: 500pt
  • Three 6: 600pt
  • Single 1: 100pt
  • Single 5: 50pt
  • Each dice can only be counted in one roll, and each roll can be only triggered once
