Password verification with regex

Regex can be useful in detecting pattern, and it can also see things out of order

In this case, a password needs to follow these rules

  • Must be eight characters long
  • Contains an uppercase letter
  • Contains a lowercase letter
  • Contains number
  • Accept only alphanumeric characters
Result

  • Look ahead
    /(?=)/
    to see if match fulfills criteria given
  • For example:
    /(?=.*[A-Z])[\w]/
    returns an true/false assertion to the match found with \w
  • /[A-Za-z0-9]${8,}/
    to test length and alphanumeric

  • Thus,
    /(?=.*[A-Z])(?=.*[a-z])(?=.*[0-9])[A-Za-z0-9]{8,}/
    will do the trick.
