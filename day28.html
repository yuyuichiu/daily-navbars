Variable Scope: var & let & const

var: function scoped (can be used within function if declared inside it)

let: block scoped (can be used within {} of where it was declared)

const: same scope with "let", but value cannot be changed.

Duplicating an array & modify it with pattern

Array.map()

Creates an new array based on target, and provides options to modify in a pattern

Syntax:   var newArr = myArr.map(x => x * 2);

That sample returns a new list with values of myArr * 2.

.sort() & Compare function

.sort()
applies to string and array. It put items in an order.

By default, it sorts the array/string by ascending order. But this behavior can be customized with compare function


Sort works by swapping values when two compared values are not in the right order

Thus here comes compare function, Compare function is a built-in function parameter for sort to determine whether two values should be swapped

It takes in two values at a time and will be called when sort() is still looping

myArr.sort( function(a, b){

   {return a - b} } );

.sort()
takes in a function as argument, which is a compare function

When the swapping happens will depend on the value returned by compare function:

  • "-ve" => [a,b] (a before b)
  • "+ve" => [b,a] (b before a, they are swapped)
  • Zero => no changes

By this rule, we can amend the value of a & b, then make it return negative or positive in our ways.
