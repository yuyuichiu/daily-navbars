Today, I deployed this project on the world wide web with a domain name

This git repository has now been sync to Netlify for online web hosting

Although the domain name is not ".com" for financial reason, it still have SSL certificate on for security.

Step 1: Sync github repository to Netlify

What is Netlify?

Netlify is a platform that provides web hosting services which supports real-time linkage with github repositories.

It provides necessary functions that you need to host the web, and allows user to setup their webpage environment.

The setup is user-friendly and easy to learn so it is easy to manage!

Netlify


Step 2: Domain Name

Getting my first domain name

Although netlify provides a free domain name of theirs, most webpage will host with their own domain name.

And, the good looking ones cost money (Fun fact: "alexa.io" cost US$94090 per year)

But the ones with ordinary words are quite assessable, usually ".com" cost around USD$8.99/year


".com" is usually the go-to choice of domain name since it is so familiar.

Since this is not a big project, I just save money and bought a cheaper domain name. (maxtest.xyz)




Step 3: Setup SSL certificate

SSL for security

SSL certificate is important for data security, search engine optimization, and to avoid browser throwing warnings to your webpage for not being safe.

Thanks to Let's Encrypt promoting network security, netlify provides free SSL certificate for verified domains.

With SSL certificate setup, data transmission are encrypted to prevent data leakage from hacking activities.
