Transition effect with dropdown list

Hover some items!

Transition Effect with Dropdown box.

It seems easy for me at first as I thought the display can be easily animated. Turns out I cannot animate properties while "display: none/ block" is adapted to manage the display. Therefore, I have to use an alternative way.

The way that I use is to use control the hover event with visibility instead of display. It was not preferable to use visibility normally as it will make the hidden items occupy spaces, but it can be solved by setting the height to 0 and overflow: hidden while the item is not shown. This way, I can add transition effect to properties.
